Краткое описание подхода к решению:
	Основная задача - свести матричную игру к алгоритму симплекс-метода решения пары 
	взаимодвойственных задач линейного программирования. Найдем сначала оптимальную 
	стратегию (P) первого игрока с помощью симплекс-метода из библиотеки scipy.linalg.optimize().
	После этого находим выигрышную стратегию (Q*) второго игрока аналогично первому.
